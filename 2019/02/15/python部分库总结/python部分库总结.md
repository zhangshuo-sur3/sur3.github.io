## 杂项隐写
### turtle
Turtle库是Python语言中一个很流行的绘制图像的函数库.
常用函数：
**setup(width,height,startx,starty) 设置窗口大小以及起始点，不设置默认为中心。**
**goto(d,d) 从当前的点指向括号内所给坐标**
**fd(d) 指沿着海龟的前方向运行**
**bk(d) 指沿着海龟的反方向运行**
**seth(angle) 改变行进方向，逆时针**
**colormode(mode) 改变画笔颜色**
**penup()/pendown() 画笔抬起/落下**
####简单应用
1.画螺旋线
![此处输入图片的描述][1]
2.猫咪遥控器

    import turtle
    
    with open('C:\Users\HP\Desktop\seq.txt') as f:
        s=f.read()
    ##print(s)
    turtle.speed(0)
    for i in s:
        if(i=='U'):
            turtle.seth(90)
            turtle.fd(1)
        elif(i=='L'):
            turtle.seth(180)
            turtle.fd(1)
        elif(i=='D'):
            turtle.seth(270)
            turtle.fd(1)
        elif(i=='R'):
            turtle.seth(0)
            turtle.fd(1)
